Essential Chrome Plugins for Developers

Published: 2021-05-10
chrome plugin developer

Note: You will need to resolve the issue of being unable to access chrome.google.com on your own.

Web Developer Assistant FeHelper (Active Users: 100k+)

  • An incredibly powerful collection of handy tools, currently including JSON自动/手动格式化, JSON内容比对, 代码美化与压缩, 信息编解码转换, 二维码生成与解码, 图片Base64编解码转换, Markdown, 网页油猴, 网页取色器, 脑图(Xmind), and more.
  • Official website: https://www.baidufe.com/fehelper/index/index.html
  • Download address: https://chrome.google.com/webstore/detail/pkgccpejnmalmdinmhkkfafefagiiiad

Axure RP Extension for Chrome (Active Users: 600k+)

  • A plugin for viewing Axure prototypes, essential for frontend and backend testing.
  • Download address: https://chrome.google.com/webstore/detail/axure-rp-extension-for-ch/dogkpdfcklifaemcdfbildhcofnopogp

uBlock Origin (Active Users: 10M+)

  • A highly efficient ad filter that supports various ad-blocking rules and updates very quickly, completely outclassing AdBlock.
  • Official website: https://ublockorigin.com/
  • Download address: https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm

Bitwarden - Free Password Manager (Active Users: 1M+)

  • A free, cross-platform, feature-rich, OTP-supported, self-hostable open-source password manager.
  • Official website: https://bitwarden.com/
  • Download address: https://chrome.google.com/webstore/detail/bitwarden-free-password-m/nngceckbapebfimnlniiiahkandclblb

Proxy SwitchyOmega (Active Users: 1M+)

  • Easily and quickly manage and switch between multiple proxy configurations. An upgrade from SwitchySharp, can replace SwitchyPlus or Proxy Switchy.
  • Download address: https://chrome.google.com/webstore/detail/proxy-switchyomega/padekgcemlokbadohgkifijomclgjgif
  • Alternative download address: https://github.com/FelisCatus/SwitchyOmega/releases

Link Grabber (Active Users: 80k+)

  • Fetch and organize all 超链接地址 on HTML页面.
  • Download address: https://chrome.google.com/webstore/detail/link-grabber/caodelkhipncidmoebgbbeemedohcdma

Forecastfox (Active Users: 10k+)

  • A 天气预报 plugin that can provide forecasts down to 每小时天气, including 体感温度 forecasts. Data sourced from: AccuWeather.com.
  • Download address: https://chrome.google.com/webstore/detail/forecastfox-fix-version/boljdehmejbffnfiiicckjhafabdepnd

React Developer Tools (Active Users: 2M+)

  • React developer tools - no explanation needed.
  • Download address: https://chrome.google.com/webstore/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi

Vue.js devtools (Active Users: 1M+)

  • Vue developer tools - no explanation needed.
  • Download address: https://chrome.google.com/webstore/detail/vuejs-devtools/nhdogjmejiglipccpnnnanhbledajbpd

Tampermonkey (Active Users: 10M+)

  • The renowned 油猴 - no explanation needed.
  • Download address: https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo

Shopping Assistant Auto Price Comparison (Active Users: 200k+)

  • Extremely convenient when browsing various shopping websites, allows viewing historical price trends for products.
  • Download address: https://chrome.google.com/webstore/detail/%E8%B4%AD%E7%89%A9%E5%85%9A%E8%87%AA%E5%8A%A8%E6%AF%94%E4%BB%B7%E5%B7%A5%E5%85%B7/jgphnjokjhjlcnnajmfjlacjnjkhleah
