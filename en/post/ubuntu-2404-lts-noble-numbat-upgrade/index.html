Ubuntu 24.04 LTS - “Noble Numbat” Upgrade Guide

Ubuntu 24.04 LTS is the version of the Ubuntu operating system released in April 2024. The development code name for this version is “Noble Numbat”.

Before you begin

You can upgrade directly to Ubuntu 24.04 LTS (“Noble Numbat”) from Ubuntu 22.04 LTS (“Jammy Jellyfish”) or Ubuntu 23.10 (“Mantic Minotaur”). After Ubuntu 24.04 LTS is released, the upgrade function will not be available immediately. Upgrading from Ubuntu 23.10 will be enabled shortly after release, at which time known upgrade issues will be resolved. Upgrades from Ubuntu 22.04 LTS will not be available until Ubuntu 24.04.1 LTS is released, which is expected to be August 2024. Make sure that all updates have been applied to your current Ubuntu version before upgrading. It is recommended to read the release notes of Ubuntu 24.04 LTS before upgrading, which records notes and workarounds for known issues in this version.

If you are using Ubuntu 22.04 LTS or a version other than 23.10, see the UpgradeNotes to learn how to upgrade.

Upgrading from 22.04 LTS or 23.10 to 24.04 LTS

Upgrade Ubuntu desktop to 24.04 LTS

You can easily upgrade over the network by following this process:

  1. Run the update-manager application.
  2. In the Update Manager, click the Settings button… and enter the password to launch the software source application.
  3. Select the “Updates” submenu in the software source application.
  4. Make sure the “Notify me of new versions of Ubuntu” option is set to “For any new version”, change it if it is different.
  5. Close the software source application and return to Update Manager.
  6. In the Update Manager, click the Check button for new updates.
  7. If there are updates that need to be installed, use the Install Updates button to install them.
  8. Run update-manager.
  9. If you want to upgrade early, before upgrading to official support, you can pass the -d option when running update-manager. Please use at your own risk. It is recommended to check the release notes for Ubuntu 24.04 LTS before doing so.

A message will appear notifying you of the availability of the new version. Click Upgrade. 10. Follow the on-screen instructions.
