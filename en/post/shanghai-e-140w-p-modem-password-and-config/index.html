Hardware configuration

  • BCM6838 chip supports the following interfaces and functions:

  1. 4 built-in Gigabit PHYs

  2. Built-in GPON MAC, EPON MAC,

  3. MIPS 32-bit processor, 400M dual core (64K×2 instruction cache, 32K data cache)

  4. Built-in voip processor

  5. 2 channels support 802.11n WLAN PCIE interface, which can be connected to PCIE AP or USB AP

  6. 2 USB 2.0 interfaces, 1 can be used as HOST, and 1 can be configured as HOST or DEVICE.

  7. DDR3 controller, supporting up to 4Gbit

  8. NAND controller, maximum support 2Gbit

  9. Security engine: supports IPSEC and SSL hardware acceleration

  10. EPON MAC + requires external PHY

  11. Built-in ARM926EJ-S*32-bit processor (16K instruction cache and 16K data cache)

  12. The Ethernet-based interface can be used to connect external VOIPSoC??

  13. Serial port: serial flash, EEPROM; parallel port: parallel flash maximum 128M
