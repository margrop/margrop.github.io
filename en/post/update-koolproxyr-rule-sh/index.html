Published: 2021-02-12
koolproxy koolproxyr rule ad

The KoolProxyR rule URLs had not been updated for a long time, so many ads could no longer be filtered. I had to change the update URLs myself.

Edit the update script

Modify the URLs

Reference

cd /koolshare/scripts
vim KoolProxyR_rule_update.sh

Modify the URL

Change the url_koolproxy field to:

url_koolproxy="http://bobohome.f3322.net:8880/files/KoolProxyRules/koolproxy.txt"

Change the url_yhosts field to:

url_yhosts="https://github.com/VeleSila/yhosts/blob/master/hosts"

Change the url_easylist field to:

url_easylist="https://raw.githubusercontent.com/uBlockOrigin/uAssets/master/filters/filters.txt"
