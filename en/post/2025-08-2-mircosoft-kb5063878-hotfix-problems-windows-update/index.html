Microsoft KB5063878 Patch May Cause Failures in Solid-State Drives (SSDs) and Hard Disk Drives (HDDs) Under Specific Circumstances

Operating System: Microsoft Windows 11 There are already many news reports on the internet about the Microsoft KB5063878 patch potentially causing failures in solid-state drives (SSDs) and hard disk drives (HDDs) under specific circumstances. For detailed information, please refer to the news links at the bottom; we won’t elaborate further here. The key question is: how do we solve this problem if we encounter it?

Recommended Method 1:

Method 1: Uninstall the patch, then disable Windows Update.

Press Win+X in the system, and click [Administrator Terminal] from the pop-up menu.

Enter: wusa /uninstall /kb:5063878, then press Enter to uninstall KB5063878

After uninstallation, press Win+I to open Settings, select Windows Update, and choose to pause updates. Wait for Microsoft to release a subsequent fix patch before re-enabling updates.

Method 2: After uninstalling the update, download the wushowhide.diagcab program, run it, select “Hide updates,” find and select KB5063878, and click Next to prevent it from appearing in the update list. Wait for Microsoft to release a subsequent fix patch before re-enabling updates.

https://download.microsoft.com/download/f/2/2/f22d5fdb-59cd-4275-8c95-1be17bf70b21/wushowhide.diagcab

Note: If the command fails to uninstall KB5063878, back up important data to an external hard drive or USB flash drive, reset the computer, and then proceed with uninstallation.
