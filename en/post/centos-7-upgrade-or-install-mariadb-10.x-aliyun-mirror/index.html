Upgrading or Fresh Installing MariaDB 10.5 on CentOS 7 Using Aliyun Mirror

Published: 2021-01-29
centos mariadb mysql yum install upgrade aliyun Aliyun mirror Aliyun repository

Update the System First

yum update -y

Edit the yum Configuration File Using Aliyun Mirror

  • Here MariaDB 10.5 is used as an example
vim /etc/yum.repos.d/MariaDB10.repo

#http://downloads.mariadb.org/mariadb/repositories/
[mariadb]
name = MariaDB
baseurl = https://mirrors.aliyun.com/mariadb/yum/10.5/centos7-amd64
gpgkey = https://mirrors.aliyun.com/mariadb/yum/RPM-GPG-KEY-MariaDB
gpgcheck = 1

Stop the Current MariaDB Service

systemctl stop mariadb

Remove Old MariaDB Installation Packages

yum remove mariadb-server mariadb mariadb-libs
yum clean all

Install MariaDB 10

yum install MariaDB-server MariaDB-client

Start and Enable Auto-Start for MariaDB

systemctl start mysql
systemctl enable mysql

Upgrade Old MariaDB Database

mysql_upgrade -uroot -p123456

Verify Installation Success

mysql --version
mysql -uroot -p123456
