Installing Docker on CentOS 7 and Configuring Aliyun Mirror Source

Published: 2021-01-29
centos docker install aliyun systemctl Aliyun container

Docker Version

Since Docker version 1.13, Docker has adopted a timeline-based versioning approach, dividing releases into Community Edition (CE) and Enterprise Edition (EE). Docker CE is the free community edition, while Docker EE is the paid enterprise version.

Uninstall and Reinstall (Optional)

sudo yum remove docker \
docker-client \
docker-client-latest \
docker-common \
docker-latest \
docker-latest-logrotate \
docker-logrotate \
docker-selinux \
docker-engine-selinux \
docker-engine

Install Required Packages

yum install -y yum-utils device-mapper-persistent-data lvm2

Add Aliyun Repository Information

cd /etc/yum.repos.d
wget http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

Check Available Docker Versions

yum list docker-ce --showduplicates | sort -r

Configure Docker Aliyun Mirror

You can configure the accelerator by modifying the daemon configuration file /etc/docker/daemon.json. In the code below, {ALIYUN_MIRROR} can be obtained by visiting 阿里云容器镜像服务 and logging into your Aliyun account.

sudo mkdir -p /etc/docker
sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://{ALIYUN_MIRROR}.mirror.aliyuncs.com"]
}
EOF
sudo systemctl daemon-reload
sudo systemctl restart docker
