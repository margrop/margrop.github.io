使用KMS激活Win10，含第三方服务和自建服务

2021-01-18 1 min read # kms # activate # slmgr # vlmcsd # gcc # firewall-cmd # 激活 # windows

第三方服务激活

  • 使用管理员权限，运行两行命令即可（鸣谢：https://03k.org/kms.html）
slmgr /skms kms.03k.org
slmgr /ato

自建服务激活

  • 以 CentOS 7为例，含完整的下载代码、编译和安装过程
  • 准备编译环境
yum install gcc git
  • 下载源代码
cd ~/tools
mkdir win
cd win
git clone https://github.com/Wind4/vlmcsd.git
  • 编译
cd vlmcsd
make
  • 启动vlmcsd服务
cd bin
./vlmcsd  
ps aux|grep vlmcsd
  • 防火墙开启1688端口
firewall-cmd --permanent --zone=public --add-port=1688/tcp
firewall-cmd --reload
  • 搞定，回到文章开开头，把 slgmr 后面的域名，改为自己服务器的域名或IP地址即可
  • 第三方服务激活
  • 自建服务激活
