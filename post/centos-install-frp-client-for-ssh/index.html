CentOS安装Frp客户端，可使用SSH

发布时间: 2021-01-23 | 标签: centos install frps frpc nohup linux client ssh

下载并解压Frp

  • 最新Frp下载地址：https://github.com/fatedier/frp/releases/
mkdir ~/frp
cd ~/frp
wget https://github.com/fatedier/frp/releases/download/v0.35.0/frp_0.35.0_linux_amd64.tar.gz
tar -xvzf frp_0.35.0_linux_amd64.tar.gz

删除服务器相关文件

cd ~/frp/frp_0.35.0_linux_amd64
rm -rf frps*

配置Frp客户端

  • 需自行修改server_addr,server_port,local_port,remote_port以及客户端名称
vi frpc.ini

[common]
server_addr = blog.margrop.net
server_port = 2000
 
[SSH-VM-CENTOS]
type = tcp
local_port = 22
remote_port = 2022

启动FRP服务

cd ~/frp/frp_0.35.0_linux_amd64
nohup ./frpc &
tail -f nohup.out
