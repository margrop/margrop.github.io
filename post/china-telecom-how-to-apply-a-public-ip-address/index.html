中国电信，如何找客服申请公网IP

发布时间: 2021-01-25 | 标签: 中国电信 telecom IP IPv4 NAS Web 公网IP 内网IP PPPoE

目前国内的三大运营商中，也就中国电信的公网IP最富裕了。 但是随着全球IPv4资源彻底耗尽，地主家也没余粮了。

最近现状

最近几年，各个地方的中国电信的家庭宽带，都偷偷摸摸的把分配的公网IP改为了10.xxx.xxx.xxx的内网IP。（中国联通和中国移动的家庭宽带就不谈了，一开始就是内网IP，而且还不能谈心） 没有了公网IP，NAS等各种服务都没法外网访问了，影响太大。

那如何找客服小姐姐谈心，要回曾经的公网IP呢？

解决方案

打10000电话或在电信营业厅APP找客服，说自己家里的摄像头无法使用了，需要公网IP。

  • 注意事项：
  1. 千万千万不要说自己需要搭建NAS或者搭建Web网站，这个做法是违反协议的。
  2. 如果客服小姐姐听不懂，那就直接让人上门服务吧；和上门的技术人员也说同样的原因，摄像头用不了。
  3. 获得外网IP后，建议拿到电信光猫的 telecomadmin的密码，然后进入光猫改为桥接，并使用自己的路由器进行PPPoE拨号，若PPPoE拨号的账号密码忘了，也可以联系客服小姐姐找回。
