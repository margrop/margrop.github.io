Windows常用软件推荐和下载，无推荐

2021-01-24 3 min read # 常用软件 # 腾讯 # 软件管家 # 应用市场 # chrome # 火绒 # zip # rar # notepad++ # unlocker # pdf # windows

这里推荐一些本站在Windows环境常用的一些软件，无推荐，不喜欢的赶紧关掉页面。

腾讯软件管家（独立版）

  • 这个软件相当于Windows版本应用市场，但是不含任何捆绑软件。
  • 腾讯官方N年前(N>3)，已经关闭了该独立版软件的下载。而为了应用市场，多安装一个腾讯电脑管家，显然是一件十分不划算的事情。
  • 腾讯同时提供了网页版本的软件中心，在Windows环境下可访问https://pc.qq.com，搜索想要的软件。
  • 访问pc.qq.com的时候，需要特别小心，别看广告下载腾讯软件管理，别点击任何高速下载，只能点普通下载，否则腾讯电脑管家就极有可能安装到你电脑里了。

Chrome 88

  • 目前使用最广泛的浏览器，没有之一。
  • 连目前国内的ZF办事部门，也慢慢在使用Chrome。

火绒安全软件

  • 其实现在这个时代，Win10的病毒防护已经做的很好了。现在根本不需要安装什么360，腾讯电脑管家，瑞星杀毒软件，金山毒霸之类的，纯粹是拖慢电脑运行速度。
  • 那为什么用这个软件呢？这个小众软件真站在用户的角度做的，在后台默默的工作，安装运行后就完全没存在感了。真有隐患时才会出现，而且内置广告屏蔽模块，对电脑上各种广告弹窗烦透的有奇效。

BandZip 6.26

  • Windows下面，没广告而又功能强大的压缩解压软件真是凤毛麟角。
  • 旧版本BandZip完美符合上面的定义，可惜最新版本的BandZip也加入了各种广告，并且不能关闭自动更新，还限制了部分功能的使用。
  • 本站提供了旧版本的BandZip下载，记得安装完成后一定要关闭自动更新。

Notepad++ 7.9.2

  • Notepad++已经在Windows下面用了好多年了，真的好用。
  • 可惜该作者最近发表了一些不好的言论，但一直习惯该软件了，没法替代，暂时先别管作者说什么了，忽略即可，重点是软件好用。

Unlocker 1.9.2

  • 很古老的一个小工具，作用也很单一，就是解决Windows有的时候文件被占用导致无法删除文件的小工具，装机必备。

云展网PDF合并工具

  • 云展网PDF合并工具是一款完全免费的多功能PDF处理工具，处理好输出的文件不带任何logo水印。
