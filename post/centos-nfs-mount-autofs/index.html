Centos 7.4 NFS自动挂载

发布时间: 2021-02-17
centos nfs autofs mount

触发式自动挂载

用户登录时自动挂载NFS共享目录/home/share到本机/mnt/share

yum install -y autofs
vi /etc/auto.master
/mnt /etc/auto.conf（这个配置文件本身就没有要自己创建）
vi /etc/auto.conf
share  192.168.1.22:/etc/letsencrypt/live/margrop.net-0001/（nfs服务器上共享文件）
service autofs restart

重启服务 将虚拟机重启并且重启服务

service autofs restart      
cd /mnt
ls 
#此时看到不到share这个目录

自己在重新建立一个

mkdir -p /mnt/share

此时

cd /mnt
ls 
#会看到share这个目录

再继续ls 会看到共享文件 df -h 查看挂载信息

永久式自动挂载

vim /etc/fstab
192.168.65.10:/home/share  /mnt/share  nfs  defaults   0     0
