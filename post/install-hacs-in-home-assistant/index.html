如何在Home Assistant里面安装HACS

发布时间: 2021-02-19 | 标签: hacs homeassistant ssh

如果是虚拟机环境（Home Assistant OS）

1.转到Supervisor面板 2.安装一个SSH附加组件（需要在用户个人资料中启用高级模式） 3.配置所选的SSH加载项 4.启动SSH附加组件 5.连接到SSH附加组件 6.运行HACS安装脚本

wget -q -O - https://hacs.xyz/install | bash -

初始配置

在您的Home Assistant用户界面中，转到配置(Configuration)，然后单击集成(Integrations) ＃ 图像

清除浏览器缓存

  • 在HACS出现在列表中之前，您需要清除浏览器缓存。

警告：您实际上需要执行此操作，不要跳过它。

点击右下角的“ +”按钮＃

搜索或向下滚动找到“HACS”，然后选择它＃

信息 当Home Assistant安装集成所需的要求时，您会看到此信息。

如果花费的时间超过10分钟，则说明Home Assistant安装可能存在问题。

接受使用

这些项目都不是可选的，您需要接受所有内容才能设置HACS。

设备注册号

  • HACS使用设备OAuth流对GitHub API进行身份验证。
  • 如果您不希望这样做，则可以在使用YAML配置HACS的情况下使用个人访问令牌。

在第一个屏幕上，向您显示设备代码，将其复制到剪贴板，然后单击https://github.com/login/device链接以继续设置。

不要点击Submit!

如果您尚未在浏览器中登录GitHub，则需要立即注册或登录以继续设置。如果已经登录，则可以跳过此部分。

加载该页面后，粘贴或键入在上一步中显示给您的设备代码。

在这里单击“授权hacs”。

看到确认屏幕后，您可以关闭选项卡并返回到Home Assistant。

现在，您可以单击“提交”以完成HACS的配置。

如果失败，将返回到“添加配置”步骤

更改配置

如果需要更改配置，则首先需要从集成列表中删除该集成，然后需要再次将其重新添加。
