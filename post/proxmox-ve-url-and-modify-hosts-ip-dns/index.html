Proxmox VE 常用本地路径，以及如何修改hosts、主机名、IP地址和DNS配置

发布时间: 2021-01-22 | 标签: pve Proxmox VE hosts ip dns network iso ovz template

PVE本地ISO文件目录：/var/lib/vz/template/iso PVE本地OVZ文件目录：/var/lib/vz/template/cache

修改hosts文件

  • 重启后生效
vi /etc/hosts

修改主机名

  • 重启后生效
vi /etc/hostname

修改IP地址和DNS配置

  • 重启后生效
vi /etc/network/interfaces

参考的DNS配置

dns-nameservers 114.114.114.114,223.5.5.5
