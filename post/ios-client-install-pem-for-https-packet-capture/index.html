iOS端如何安装证书，如何https抓包

发布时间: 2021-01-28 | 标签: ios pem cert https 抓包 packet capture charles web safari 描述文件

iOS下载证书

  • 在Charles中导出pem证书，放置任意Web服务器中
  • 下面是我自己使用的pem证书，仅作为示例，请不要安装。
  • 用iOS Safari打开URL，下载pem证书，iOS会提示是否下载描述文件，下载即可

安装描述文件

  • 描述文件安装完成后，打开iOS的设置，会看到多出一行，已下载描述文件。
  • 选择刚下载的Charles的描述文件，点击右上角的安装
  • 输入锁屏密码，然后再次点击右上角的安装，即可完成安装过程

信任证书

  • 再回到iOS的设置 -> 通用 -> 关于本机 -> 证书设置，启用Charles证书。
