OpenCore 的 config.plist 文件，如何打印日志？

2021-12-14 1 min read

准备工作

  • PlistEdit Pro
  • config.plist

开始

  1. 使用PlistEdit Pro 打开 config.plist文件
  2. 展开NVRAM -> ADD -> 7C436110-AB2A-4BBB-A880-FE41995C9F82
  3. 修改boot-args参数，在最末尾添加 -v
