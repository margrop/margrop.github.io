Vim打开文件中文乱码

2021-05-15 1 min read # vim # Linux

解决办法

  • 编辑 ~/.vimrc 文件（没有则手动创建） 添加如下几行
set fileencodings=utf-8,ucs-bom,gb18030,gbk,gb2312,cp936
set termencoding=utf-8
set encoding=utf-8
