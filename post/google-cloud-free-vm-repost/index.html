之前介绍了几个免费的云服务器，那其实海外三大云服务商（谷歌、亚马逊、微软）都是有自己的免费服务器，只是后面两个都是限时，不好玩，谷歌除了能试用三个月正价机器，还有个永久免费的小鸡，2核1g内存30g硬盘，200g流量/月，玩一玩还是可以的。
一、谷歌云&试用&免费
谷歌云，顾名思义就是谷歌做的云服务器。现在谷歌云的试用是只有三个月，好像是300美元额度，买正价机器一下子就用完了，所以，不如玩免费服务器。
本篇不涉及谷歌云注册以及试用那些，这玩意有谷歌账号的都能注册，没啥难度，而且我好几年前就注册过了，具体细节早就不记得，不知道的自行谷歌，就算是新账户还在试用中也不影响本文试用免费实例。
主要是谷歌云的免费服务器，谷歌云给了对应地区的免费配额，俄勒冈、爱荷华、南卡罗来纳的e2-micro实例，2核cpu+1g内存+30g硬盘，带公网ipv4，每个月200g流量，这个流量是不限地区的，有人会说图片内不是写着“每月 1 GB出站流量吗”，是的，首先谷歌云是只计算出站流量，而非双向流量，其次图内的这个免费1g流量是“高级层级”，而非“标准层级”的流量，高级层级要比标准层级网络质量好一点，当然也仅限于好一点（虽然以前不是），而标准层级是全球200g流量。
但有一点是例外，akari、cloudflare、fastly这三家cdn厂商是和谷歌有合作，连接到他们的流量是要钱的，所以我们这个服务器是不能连接这三家cdn。我知道很多人cloudflare都不能连，还能干啥，有一说一，确实是会限制可玩性，但你当个个人小型服务器，例如签到、自建笔记这些不都行，玩法也是有的。
二、建立免费实例
登录谷歌云，创建虚拟机，选择俄勒冈地区，当然其他三个区域也可以，但是延迟要比俄勒冈高，没必要选。实例规格选择e2-micro，硬盘一定要选择“新的标准永久性磁盘”，只有这个才免费，大小30g，镜像我建议选debian13，关闭数据保护，备份也是要花钱的，不关等着被反薅。
网络这里可以先配置防火墙，允许80和443端口，当然你这里不开也行，后面去控制台防火墙配置也行，默认是放行了22、3389端口以及icmp。但还需要修改网络接口，默认ipv4单栈，ipv6是要钱的。公网ipv4选择临时，这个意思是停止后ipv4会变动，不是固定的，那为什么不固定呢？要钱，但只要服务器不停止，这个ip是不会变的。网络服务层级选择标准，这样就可以每个月200g免费流量。没啥要求记得把可观测性也给取消了，这玩意会给服务器按照额外的插件，没啥用。最后确认无误创建即可。
有人会问，旁边有估算费用，要不话花钱。。。。当然是不用花钱啊，要花钱我还说什么免费服务器。当然你怕花钱，那就时不时盯着账单，如果有花费立马取消实例就行，我只能说我目前是没被扣费，因为也有人偶尔会每个月有个零点几美元的扣费，但基本上是不小心连接到那三家cdn导致的，不行就给卡限额。对了，实例建好后千万不要开启虚拟机管理器，这玩意也是要花钱的.....
三、配置防火墙
上面说了，这个免费服务器是不能连接akari、cloudflare、fastly这三家cdn，不然会算流量钱，所以想要真免费用这个服务器，我们还需要通过配置防火墙禁止服务器连接这三家cdn的ip，实现定向封锁。一般情况下cdn厂商的ip是不会变动的，像cloudflare都已经好几年没变动ip了，所以只要配置好阻止访问对应ip基本上就不用担心。
常规是直接在谷歌云控制台内配置防火墙，但是那样太麻烦了，好在某论坛大佬制作了iptables配置脚本，只需要在系统内运行即可阻止对应ip。当然如果觉得还是控制台内配置防火墙也可以，自行收集对应cdn的ip地址即可，官网都有的。
进入云服务器控制台，新建的服务器一开始是只能在控制台ssh登录，依次输入以下命令，完成配置，特别提醒，一定一定要先更改软件源，因为debian默认源是走cloudflare，得花钱。
sudo passwd#配置root密码，默认不是root用户且无密码
su -#切换root用户
sudo sed -i 's/^#*PermitRootLogin.*/PermitRootLogin yes/' /etc/ssh/sshd_config && sudo systemctl restart ssh#启用root ssh登录，如果不需要可以不执行这个
sudo bash -c '\rm -rf /etc/apt/sources.list.d/* && \truncate -s 0 /etc/apt/sources.list && \cat > /etc/apt/sources.list <<EOFdeb http://mirrors.aliyun.com/debian/ bookworm main contrib non-free non-free-firmwaredeb http://mirrors.aliyun.com/debian/ bookworm-updates main contrib non-free non-free-firmwaredeb http://mirrors.aliyun.com/debian-security bookworm-security main contrib non-free non-free-firmwaredeb http://mirrors.aliyun.com/debian/ bookworm-backports main contrib non-free non-free-firmwareEOFapt update'#更换阿里源，我这是debian，其他系统自行更换#一定要先更换源，国内源不走那三个cdn不花钱
apt install -y iptables ipset curl#安装iptables防火墙
bash <(curl -fsSL https://quzei.com/sh/cdn_block.sh)#执行大佬的脚本，把阻止ip列表写入iptables
目前我这个也使用了一段时间，并没发现有扣费，这个ip封锁列表应该是完整的，当然如果你担心不全，自己收集整理也可以。我也是ping了下这个服务器的延迟，能用就行了，免费的还要啥自行车。
