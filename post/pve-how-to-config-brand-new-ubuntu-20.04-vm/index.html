修改apt-get为aliyun源：

  • 用你熟悉的编辑器打开/etc/apt/sources.list，编辑为下面的内容

安装基本网络工具

sudo apt install net-tools

修改root密码为vm

输入 sudo passwd 命令，然后会提示输入当前用户的密码。
按enter键，终端会提示输入新的密码并确认，此时的密码就是新的root密码。
修改完毕以后，在执行su root命令，此时输入新的root密码即可。
