更新 KoolProxyR 的规则地址，2021年2月12日更新

发布时间: 2021-02-12 | 标签: koolproxy koolproxyr rule ad

KoolProxyR 的规则地址已经好久没更新了，好多广告都没法过滤 只能自己改下更新地址了

编辑更新脚本

cd /koolshare/scripts
vim KoolProxyR_rule_update.sh

修改URL

url_koolproxy字段改为

url_koolproxy="http://bobohome.f3322.net:8880/files/KoolProxyRules/koolproxy.txt"

url_yhosts字段改为

url_yhosts="https://github.com/VeleSila/yhosts/blob/master/hosts"

url_easylist字段改为

url_easylist="https://raw.githubusercontent.com/uBlockOrigin/uAssets/master/filters/filters.txt"

参考文章 https://www.right.com.cn/forum/thread-3770004-1-1.html https://github.com/VeleSila/yhosts https://github.com/fang5566/uBlock/blob/master/README.md#ublock-origin
