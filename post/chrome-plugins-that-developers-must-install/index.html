开发者必备Chrome插件

2021-05-10 3 min read # plugin # developer # chrome

注：请自行解决无法访问chrome.google.com的问题

Web开发者助手 FeHelper（使用人数，10w+）

  • 超级强大的小工具集合，目前包括 JSON自动/手动格式化、JSON内容比对、代码美化与压缩、信息编解码转换、二维码生成与解码、图片Base64编解码转换、Markdown、 网页油猴、网页取色器、脑图(Xmind)等贴心工具
  • 官方网站：https://www.baidufe.com/fehelper/index/index.html
  • 下载地址：https://chrome.google.com/webstore/detail/pkgccpejnmalmdinmhkkfafefagiiiad

Axure RP Extension for Chrome（使用人数，60w+）

  • 浏览Axure原型的插件，前端后端测试必备
  • 下载地址：https://chrome.google.com/webstore/detail/axure-rp-extension-for-ch/dogkpdfcklifaemcdfbildhcofnopogp

uBlock Origin（使用人数，1000w+）

  • 一款非常高效的广告过滤器，支持各种广告过滤规则，更新非常快，完爆AdBlock
  • 官方网站：https://ublockorigin.com/
  • 下载地址：https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm

Bitwarden - Free Password Manager（使用人数：100w+）

  • 一款免费的，跨各种平台的，功能强大的，支持OTP的，可自行搭建服务器的开源密码管理器
  • 官方网站：https://bitwarden.com/
  • 下载地址：https://chrome.google.com/webstore/detail/bitwarden-free-password-m/nngceckbapebfimnlniiiahkandclblb

Proxy SwitchyOmega（使用人数：100w+）

  • 轻松快捷地管理和切换多个代理设置，。是SwitchySharp的升级版，可替代SwitchyPlus或 Proxy Switchy
  • 下载地址：https://chrome.google.com/webstore/detail/proxy-switchyomega/padekgcemlokbadohgkifijomclgjgif
  • 备用下载地址： https://github.com/FelisCatus/SwitchyOmega/releases

Link Grabber（使用人数：8w+）

  • 获取并整理 HTML页面上面的所有超链接地址
  • 下载地址：https://chrome.google.com/webstore/detail/link-grabber/caodelkhipncidmoebgbbeemedohcdma

Forecastfox (使用人数：1w+)

  • 天气预报插件，能精确到每小时天气，还有体感温度预报，数据源来自：AccuWeather.com
  • 下载地址：https://chrome.google.com/webstore/detail/forecastfox-fix-version/boljdehmejbffnfiiicckjhafabdepnd

React Developer Tools（使用人数：200w+）

  • React 开发者工具，无需解释
  • 下载地址：https://chrome.google.com/webstore/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi

Vue.js devtools（使用人数：100w+）

  • Vue 开发者工具，无需解释
  • 下载地址：https://chrome.google.com/webstore/detail/vuejs-devtools/nhdogjmejiglipccpnnnanhbledajbpd

Tampermonkey（使用人数：1000w+）

  • 大名鼎鼎的油猴，无需解释
  • 下载地址：https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo

购物党自动比价工具（使用人数：20w+）

  • 在各大购物网站时非常方便，能查看商品的历史价格曲线
  • 下载地址：https://chrome.google.com/webstore/detail/购物党自动比价工具/jgphnjokjhjlcnnajmfjlacjnjkhleah
