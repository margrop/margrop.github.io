PVE因boot空间不足而升级失败的解决办法
2021-01-17 2 min read # boot # kernel # pve # Proxmox VE # pvekclean

默认安装时，proxmox给/boot分配的空间才不到300M。而proxmox的迭代很快，/boot很容易就被占满，稍不注意就导致升级失败，唯一的做法就是手工删除旧的kernel。
